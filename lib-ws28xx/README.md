# Library WS28xx
## Open Source C++ library for the WS28xx / SK6812 / APA102 / UCSx903 / P9813 chips

The supported chips are:

- WS2801
- WS2811
- WS2812
- WS2812B
- WS2813
- WS2815
- SK6812 (RGBW)
- APA102
- UCS1903
- UCS2903
- P9813
