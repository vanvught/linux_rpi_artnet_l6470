# Library Display
## Cross-platform Open Source C++

Supported displays:

- I2C
	- OLED text display 20x4 SSD1311
	- OLED display 128x32 SSD1306
	- OLED display 128x64 SSD1306 
	- LCD Module TC1602A / PCF8574T 
	- LCD Module 2004 / PCF8574T
- SPI
	- ST7789 240x240
	- ST7789 240x320
